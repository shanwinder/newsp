# แผนการพัฒนาระบบ: การเปลี่ยนชื่อและอัตลักษณ์ของสื่อ

## 1. ชื่อแผนงาน

**แผนการปรับชื่อระบบและอัตลักษณ์สื่อจาก “Young Smart Farmer” เป็น “เกมแบบฝึกทักษะออนไลน์ เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน”**

---

## 2. วัตถุประสงค์

เพื่อปรับชื่อ ข้อความ และอัตลักษณ์ที่ปรากฏในระบบให้สอดคล้องกับชื่อนวัตกรรมใหม่ คือ

> **การพัฒนาเกมแบบฝึกทักษะออนไลน์ เพื่อส่งเสริมทักษะการแก้ปัญหาอย่างเป็นขั้นตอน เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน โดยใช้การจัดการเรียนรู้แบบใช้เกมเป็นฐาน (Game-Based Learning) สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4**

โดยให้ระบบมีชื่อที่เหมาะสมต่อการใช้งานจริงในหน้าจอ และยังคงเชื่อมโยงกับชื่อนวัตกรรมฉบับเต็มที่ใช้ในเอกสารรายงานผลงานและการเผยแพร่ใน OBEC Content Center

---

## 3. ขอบเขตของแผนนี้

แผนนี้ครอบคลุมเฉพาะ **การเปลี่ยนชื่อและอัตลักษณ์ข้อความในระบบ** เท่านั้น ได้แก่

- ชื่อระบบ
- ชื่อสั้นของระบบ
- ชื่อเต็มของนวัตกรรม
- คำอธิบายใต้ชื่อระบบ
- ชื่อหน้าเว็บไซต์
- ชื่อที่ปรากฏใน Navbar
- ชื่อที่ปรากฏในหน้า Login
- ชื่อที่ปรากฏใน Dashboard
- ชื่อที่ปรากฏในหน้าคู่มือ/คำแนะนำ
- ข้อความที่เกี่ยวข้องกับธีม Young Smart Farmer

แผนนี้ **ยังไม่ครอบคลุม** การพัฒนา Multi-school, ระบบสมัครสมาชิก, การปรับฐานข้อมูล, การเพิ่มบทเรียน, การเขียนรายงานผลงาน หรือการนำขึ้น OBEC Content Center

---

## 4. หลักการตั้งชื่อใหม่

เนื่องจากชื่อผลงานเต็มมีความยาวมาก จึงไม่ควรนำไปแสดงตรง ๆ ในทุกหน้าของระบบ เพราะจะทำให้หน้าจอแน่น อ่านยาก และไม่เหมาะกับนักเรียนระดับประถมศึกษา

จึงกำหนดชื่อเป็น 4 ระดับ ดังนี้

| ระดับชื่อ | ใช้ในส่วนใด | ข้อความที่กำหนด |
|---|---|---|
| ชื่อเต็มของนวัตกรรม | เอกสารรายงาน, หน้าเกี่ยวกับสื่อ, OBEC Content Center | การพัฒนาเกมแบบฝึกทักษะออนไลน์ เพื่อส่งเสริมทักษะการแก้ปัญหาอย่างเป็นขั้นตอน เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน โดยใช้การจัดการเรียนรู้แบบใช้เกมเป็นฐาน (Game-Based Learning) สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4 |
| ชื่อระบบหลัก | หน้าแรก, Login, Dashboard, Navbar | เกมแบบฝึกทักษะออนไลน์ |
| คำอธิบายระบบ | ใต้ชื่อระบบ, หน้าแนะนำ, หน้าเลือกบทเรียน | เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4 |
| ชื่อธีมภายในเกม | ภารกิจ, ด่าน, ข้อความสร้างแรงจูงใจ | ภารกิจฟาร์มแก้ปัญหา |

---

## 5. แนวทางการใช้คำว่า Young Smart Farmer

คำว่า **Young Smart Farmer** ไม่ควรเป็นชื่อหลักของระบบอีกต่อไป แต่สามารถเก็บไว้เป็น “ชื่อธีมเดิม” หรือ “ชื่อแนวคิดด้านฉากเกม” ได้ หากยังต้องการรักษาบรรยากาศเกมฟาร์มไว้

แนวทางที่แนะนำคือ

| ข้อความเดิม | แนวทางใหม่ |
|---|---|
| Young Smart Farmer | เกมแบบฝึกทักษะออนไลน์ |
| ศูนย์ฟาร์มอัจฉริยะ | ศูนย์ฝึกทักษะการแก้ปัญหา |
| เข้าสู่ฟาร์ม | เข้าสู่บทเรียน |
| เลือกแปลงเกษตรของคุณ | เลือกภารกิจการเรียนรู้ |
| สุดยอด Young Smart Farmer | นักแก้ปัญหาอย่างเป็นขั้นตอน |
| แฟ้มภารกิจ Young Smart Farmer | แฟ้มภารกิจการแก้ปัญหา |
| เกษตรกรฝึกหัด | นักคิดเริ่มต้น |
| ฟาร์มเกษตรกรน้อย | ห้องเรียนเกมแบบฝึกทักษะออนไลน์ |

ทั้งนี้ คำที่เกี่ยวกับฟาร์มยังสามารถใช้ได้ในบริบทของเกม เช่น แปลงเกษตร รถไถ เครื่องรดน้ำ หรือฟาร์มอัจฉริยะ แต่ไม่ควรใช้เป็นชื่อหลักของระบบ

## 7. โครงสร้างชื่อกลางของระบบ

เพื่อไม่ให้ต้องแก้ชื่อซ้ำหลายไฟล์ ควรสร้างไฟล์ตั้งค่ากลางสำหรับชื่อระบบ เช่น

```text
config/app.php
```

ตัวอย่างโครงสร้างที่แนะนำ

```php
<?php

return [
    'app_name' => 'เกมแบบฝึกทักษะออนไลน์',
    'app_subtitle' => 'เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4',
    'innovation_title' => 'การพัฒนาเกมแบบฝึกทักษะออนไลน์ เพื่อส่งเสริมทักษะการแก้ปัญหาอย่างเป็นขั้นตอน เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน โดยใช้การจัดการเรียนรู้แบบใช้เกมเป็นฐาน (Game-Based Learning) สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4',
    'theme_name' => 'ภารกิจฟาร์มแก้ปัญหา',
    'subject_area' => 'วิทยาการคำนวณ',
    'grade_level' => 'ชั้นประถมศึกษาปีที่ 4',
    'short_grade' => 'ป.4',
];
```

จากนั้นให้ทุกหน้าเรียกใช้ค่าจากไฟล์นี้แทนการเขียนชื่อระบบซ้ำเอง

ตัวอย่างการเรียกใช้

```php
<?php
$app = require __DIR__ . '/../config/app.php';
?>

<title><?php echo htmlspecialchars($app['app_name']); ?></title>
```

## 8. แผนการดำเนินงาน

### ขั้นที่ 1 สำรวจข้อความเดิมในระบบ

ค้นหาคำที่เกี่ยวข้องกับชื่อเดิมทั้งหมด เช่น

```text
Young Smart Farmer
ฟาร์ม
เกษตรกร
แปลงเกษตร
ศูนย์ฟาร์ม
เข้าสู่ฟาร์ม
เกษตรกรน้อย
```

แล้วบันทึกว่าแต่ละคำอยู่ในไฟล์ใด ใช้ในบริบทใด และควรเปลี่ยนหรือคงไว้

ผลลัพธ์ที่ต้องได้จากขั้นนี้คือ

- รายการไฟล์ที่ต้องแก้
- รายการคำเดิม
- รายการคำใหม่
- บริบทที่ควรคงธีมฟาร์มไว้

### ขั้นที่ 2 กำหนดมาตรฐานชื่อใหม่

กำหนดชื่อใหม่ให้ชัดก่อนเริ่มแก้โค้ด เพื่อป้องกันการใช้ชื่อไม่ตรงกันในแต่ละหน้า

มาตรฐานที่ใช้คือ

```text
ชื่อระบบหลัก: เกมแบบฝึกทักษะออนไลน์
คำอธิบายระบบ: เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4
ชื่อธีมเกม: ภารกิจฟาร์มแก้ปัญหา
ชื่อผู้เรียนในระบบ: นักแก้ปัญหา
ชื่อคะแนน/ดาว: ดาวภารกิจ
```

### ขั้นที่ 3 สร้างไฟล์ตั้งค่ากลาง

สร้างไฟล์ `config/app.php` เพื่อเก็บชื่อระบบทั้งหมดไว้ในที่เดียว

งานที่ต้องทำ

- สร้างโฟลเดอร์ `config` หากยังไม่มี
- สร้างไฟล์ `app.php`
- เพิ่มตัวแปรชื่อระบบตามโครงสร้างในข้อ 7
- ทดสอบว่าไฟล์สามารถ require/include ได้จากทุกหน้า

### ขั้นที่ 4 ปรับหน้าแรกและหน้า Login

ปรับชื่อใน `index.php` และ `pages/login.php` ก่อน เพราะเป็นหน้าที่ผู้ใช้เห็นเป็นอันดับแรก

สิ่งที่ต้องปรับ

- `<title>`
- หัวข้อหลัก
- คำโปรย
- ปุ่มเข้าสู่ระบบ
- ข้อความต้อนรับ
- ข้อความเลือกโหมดการเรียนรู้

ตัวอย่างข้อความใหม่

```text
ยินดีต้อนรับสู่เกมแบบฝึกทักษะออนไลน์
เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4
เรียนรู้ผ่านภารกิจเกม สนุก คิดเป็นขั้นตอน และแก้ปัญหาอย่างมีเหตุผล
```

### ขั้นที่ 5 ปรับ Navbar และ Dashboard นักเรียน

ปรับ `includes/student_navbar.php` และ `pages/student_dashboard.php`

สิ่งที่ต้องปรับ

- ชื่อระบบใน Navbar
- คำอธิบายใต้ชื่อ
- ข้อความแสดงคะแนน
- หัวข้อหน้าเลือกบทเรียน
- ปุ่มเข้าสู่บทเรียน
- ข้อความฉายานักเรียน

ตัวอย่างข้อความใหม่ใน Navbar

```text
เกมแบบฝึกทักษะออนไลน์
การแก้ปัญหาอย่างเป็นขั้นตอน ป.4
```

ตัวอย่างข้อความใหม่ใน Dashboard

```text
เลือกภารกิจการเรียนรู้
สะสมดาวภารกิจเพื่อเป็นนักแก้ปัญหาอย่างเป็นขั้นตอน
```

### ขั้นที่ 6 ปรับหน้าคู่มือและหน้าเลือกด่าน

ปรับ `pages/instruction.php` และ `pages/game_select.php`

สิ่งที่ต้องปรับ

- ชื่อแฟ้มภารกิจ
- คำอธิบายภารกิจ
- ปุ่มเริ่มเรียน
- ปุ่มย้อนกลับ
- ข้อความผ่านด่าน
- ข้อความสร้างชิ้นงานหลังผ่านครบทุกด่าน

ตัวอย่างข้อความใหม่

```text
แฟ้มภารกิจการแก้ปัญหา
อ่านคำแนะนำให้เข้าใจ แล้วเริ่มฝึกคิดอย่างเป็นขั้นตอน
```

### ขั้นที่ 7 ปรับหน้าเล่นเกมและหน้าสร้างชิ้นงาน

ปรับ `pages/play_game.php` และหน้าสร้างชิ้นงาน เช่น `pages/create_project_logic.php`

สิ่งที่ต้องปรับ

- title หน้าเว็บ
- ปุ่มออกจากด่าน
- ข้อความเมื่อเล่นสำเร็จ
- ข้อความสร้างชิ้นงาน
- ข้อความ feedback
- คำว่า “ฟาร์ม” ที่ใช้เป็นชื่อระบบหลัก

แนวทางคือให้คงคำว่า “ฟาร์ม” เฉพาะในบริบทของโจทย์เกม เช่น “แปลงผัก”, “รถไถ”, “เครื่องรดน้ำ” แต่เปลี่ยนข้อความที่เป็นชื่อระบบให้เป็น “เกมแบบฝึกทักษะออนไลน์”

### ขั้นที่ 8 ปรับ Dashboard ครู

ปรับ `pages/dashboard.php`

สิ่งที่ต้องปรับ

- title หน้า Teacher Dashboard
- ชื่อระบบฝั่งครู
- หัวข้อควบคุมการเข้าเรียน
- หัวข้อควบคุมด่าน
- ปุ่มตรวจชิ้นงาน
- ข้อความเกี่ยวกับนักเรียนและคะแนน

ตัวอย่างข้อความใหม่

```text
Teacher Dashboard - เกมแบบฝึกทักษะออนไลน์
สถานะการเข้าเรียน
การปลดล็อกภารกิจถัดไป
จัดการนักเรียนและผลการเรียนรู้
```

### ขั้นที่ 9 ปรับข้อมูลในฐานข้อมูล

ปรับข้อมูลในตารางที่แสดงชื่อเกมหรือฉายา เช่น

- `games.title`
- `games.description`
- `games.learning_topic`
- `games.instruction_html`
- `titles.title_name`

แนวทางชื่อหน่วยการเรียนรู้ที่ควรใช้

| ลำดับ | ชื่อหน่วย | ทักษะที่เน้น |
|---|---|---|
| 1 | ตรรกะคัดแยก | การคิดเชิงตรรกะและการจำแนกเงื่อนไข |
| 2 | เส้นทางเดินรถไถ | การเรียงลำดับขั้นตอน |

## 10. เกณฑ์การตรวจสอบหลังเปลี่ยนชื่อ

หลังแก้ไขเสร็จ ต้องตรวจสอบตามรายการต่อไปนี้

- [ ] ไม่มีคำว่า Young Smart Farmer ปรากฏในฐานะชื่อหลักของระบบ
- [ ] ชื่อระบบหลักแสดงเป็น “เกมแบบฝึกทักษะออนไลน์”
- [ ] คำอธิบายระบบแสดงเป็น “เรื่อง การแก้ปัญหาอย่างเป็นขั้นตอน สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4”
- [ ] หน้าแรกใช้ชื่อและคำอธิบายถูกต้อง
- [ ] หน้า Login ใช้ชื่อและคำอธิบายถูกต้อง
- [ ] Navbar นักเรียนใช้ชื่อใหม่
- [ ] Dashboard นักเรียนใช้ชื่อใหม่
- [ ] Dashboard ครูใช้ชื่อใหม่
- [ ] หน้าคู่มือใช้ชื่อใหม่
- [ ] หน้าเลือกด่านใช้ชื่อใหม่
- [ ] หน้าเล่นเกมใช้ชื่อใหม่

## 13. ผลลัพธ์ที่คาดหวัง

เมื่อดำเนินการตามแผนนี้เสร็จ ระบบจะมีอัตลักษณ์ใหม่ที่ชัดเจน ดังนี้

- ชื่อระบบสอดคล้องกับชื่อนวัตกรรม
- หน้าจอระบบเหมาะกับนักเรียนชั้นประถมศึกษาปีที่ 4
- ชื่อไม่ยาวเกินไปสำหรับการใช้งานจริง
- ยังสามารถใช้ชื่อเต็มในรายงานผลงานและ OBEC Content Center ได้
- ธีมฟาร์มยังคงอยู่ในฐานะบริบทของเกม ไม่ใช่ชื่อหลักของระบบ
- เอกสารรายงาน คู่มือ และระบบจริงจะใช้ชื่อไปในทิศทางเดียวกัน

## 14. Definition of Done

ถือว่างานเปลี่ยนชื่อเสร็จสมบูรณ์ เมื่อผ่านเงื่อนไขต่อไปนี้

1. มีไฟล์ตั้งค่ากลางสำหรับชื่อระบบ
2. ทุกหน้าหลักเรียกใช้ชื่อจากไฟล์ตั้งค่ากลาง
3. ไม่มีชื่อ Young Smart Farmer ปรากฏเป็นชื่อหลักของระบบ
4. ชื่อระบบใหม่ปรากฏถูกต้องทุกหน้า
5. ชื่อบทเรียน 4 หน่วยตรงกันในระบบ
6. ฐานข้อมูลและหน้าจอใช้คำเรียกเดียวกัน
7. ทดสอบการแสดงผลบนคอมพิวเตอร์ แท็บเล็ต และมือถือแล้วไม่พบปัญหาชื่อยาวล้นจอ
8. มีภาพหน้าจอหลังปรับชื่อเพื่อใช้เป็นหลักฐานการพัฒนา
